# Features

*k*UPS is a comprehensive Monte Carlo simulation package designed for molecular systems. This section provides an overview of the key features and capabilities available in *k*UPS.

## Overview

*k*UPS provides a flexible and efficient framework for Monte Carlo simulations with support for:

- **Multiple Simulation Ensembles**: Support for NVT and GCMC simulations
- **Advanced Monte Carlo Moves**: Optimized moves for efficient sampling
- **Flexible Force Fields**: Support for classical and machine learning potentials

## Feature Categories

### [Simulation Ensembles](ensembles.md)
Learn about the different statistical ensembles supported by *k*UPS, including canonical (NVT) and grand canonical (GCMC) simulations.

### [Monte Carlo Moves](mc_moves.md)

### [Force Fields](force_fields.md)

## Current Development Status

> *k*UPS is currently focused on rigid-body molecules. Future releases will expand support to flexible molecules and additional simulation types.
